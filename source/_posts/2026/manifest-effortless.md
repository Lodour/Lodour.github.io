最近意识到自己有一个根深蒂固的限制性信念：

{% cq %}

获得 XX 是一件很难的事情。

{% endcq %}

这里的 XX 可以是理想的工作、美好的生活、健康的亲密关系等等。

正是因为觉得这件事很难，所以潜意识里会认为自己需要很用力、很小心、很完美才能获得某个东西，于是非常想要控制一切、保证一切都处于安全的范围，纠结每一件小事要怎么才能做到最好，并且非常渴求证明自己、得到别人的认同，然后害怕别人的离开或者失去某些东西。

而这种恐惧，就是一切负面情绪的根源，从而在现实层面实现了「自己很难获得某样东西」的信念。当我释放掉这个限制性信念以后，发现内心的想法开始改变了。

我允许自己轻松地获得，所以我完全不会想到要去尝试控制外界事情的状态和运行方式，也不用担心是不是得到了别人的认可。我只需要遵从自己的本心，如果别人不认可或者不满意，那别人直接离开就好了，有什么需要担心的呢？正是因为这件事很轻松，所以我不再害怕目光所及的事物不稳定、可能失去、想要离开，因为只要我愿意，马上就会有新的、轻松的机会出现在我身边，然后我再轻松地去做我需要做的事情，没有任何需要担忧的事情。

对吧，本来就没有要担心的事情。

<!--more-->

至于如何释放掉这个限制性信念：

首先在纸上写下「我允许自己轻松地获得 XX」这个目标。

然后静下心来感受这个目标，想象这个目标是真的 / 假的时候所浮现的感受。

假设这个目标是真的实现了，浮现出来的感受可能是多面的：

* 那就太好了！放松、开心、喜悦、平静、安全、可控、得到认可 ...
* 我真的可以吗？怀疑、自卑、不自信、逃避、恐惧 ...
* 如果事实不轻松怎么办？恐惧、担忧、绝望、焦虑、想要控制 ...
* 经验告诉我这不可能。徒劳、懒惰、悲苦、恐惧、渴求、委屈 ...

假设这个目标永远无法实现，浮现出来的感受同样是多面的：

* 那还是太好了！可以逃避、避免恐惧、想要安全、想要控制 ...
* 果然没那么容易。恐惧、悲苦、否定、愤怒、放弃 ...
* 那还是别努力了。放弃、绝望、徒劳、想要安全、想要控制 ...
* 可以不用担心失败了。逃避、徒劳、想要安全、想要控制 ...

最后把这些情绪全部都释放掉，然后不断问自己：

{% cq %}

我真的、真的、注意是真的，允许自己轻松地获得 XX 吗？

{% endcq %}

然后某个瞬间会突然发现，这似乎确实是一件很轻松的事情。

然后就放下了所有的担忧和阻力。
